# 蜂后与工蜂协议示例

## 1. 目标

本规范给出协议层的示例消息，便于后续实现和测试。

## 2. 建议包含的示例消息

- `hello`
- `hello_ack`
- `heartbeat`
- `task_assign`
- `task_progress`
- `task_result` 成功示例
- `task_result` 失败示例
- `shutdown`
- 拒绝连接示例

## 3. 示例用途

这些示例可用于：

- 协议实现对照
- 测试样例
- 回放样例
- 调试与日志校验

## 4. 第一版建议

第一版优先把消息结构、方向和校验关系定稳，不必过早增加复杂消息类型。
